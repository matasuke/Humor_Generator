本実験では，画像を下に文章を自動生成するシステムの評価を行っていただきます。 システムが画像を下にいくつかの文章を自動生成するので，生成された文章に対して，ユーモアを感じたか，面白いと感じたかどうか評価していただきます。 各画像に対して5つの文章が提示されるので，それぞれの文章に対して，1点から5点で面白さを評価してください。 画像は全部で{{ exp_num }}枚提示されます。

{% endblock %} {% block experiment_flow %}
  1. 左に表示される画像を見てください。
  2. 右に表示される文章を見てください。
  3. 提示された文章の面白さを1点から5点で評価してください。
  4. 「次へ」ボタンを押して，結果を送信してください。
  5. 同じ画像に対して別の文章が表示されます。
  6. 1~5を5回繰り返すと，次の画像へ進みます。
  7. 上部のプロセスバーが{{ exp_num*4 }}/{{ exp_num*4 }}になったら終了です。
  8. 「次へ」を押して実験を終了してください。
{% endblock %} {% block experiment_caution %}
  • なるべくリラックスできる環境で評価してください。
  • 評定内容は深く考えず選択してください。
