本実験では，画像を下に文章を自動生成するシステムの評価を行っていただきます。 システムが画像を下に，発話内容を自動的に生成します。 AppleのSiriやSoftbankのPepperなどといった対話システムを利用しているという仮定の下，生成された発話内容を見た際，もっと利用してみたいと思ったかどうか，もっと話してみたいと思ったかどうか1点から5点で評価してください。 各画像に対して3つの文章が提示されます． 画像は全部で{{ exp_num }}枚提示されます。

{% endblock %} {% block experiment_flow %}
  1. 左に表示される画像を見てください。
  2. 右に表示される文章を見てください。
  3. 提示された文章の面白さを1点から5点で評価してください。
  4. 「次へ」ボタンを押して，結果を送信してください。
  5. 同じ画像に対して別の文章が表示されます。
  6. 1~5を3回繰り返すと，次の画像へ進みます。
  7. 上部のプロセスバーが{{ exp_num*3 }}/{{ exp_num*3 }}になったら終了です。
  8. 「次へ」を押して実験を終了してください。
{% endblock %} {% block experiment_caution %}

  • 評定内容は深く考えず選択してください。
